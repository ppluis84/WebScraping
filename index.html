Bot para la extracción de datos de la web (Python - Web Scraping)

Enero de 2025

Resumen:

El objetivo de este proyecto es utilizar el método web scraping para la extracción de datos de la web, la librería utilizada es selenium de Python.


Librerias utilizadas

Las librerías utilizadas son SELENIUM, de python y sus dependencias que sirven para la navegación del bot en las páginas web TIME para el control de los tiempos de espera, ChromeDriverManager para la apertura de Chrome y webdriver.Chrome el control de la navegación mediante el bot.

Código que scrapea la web

Se crean 3 listas vacías en las que se va a guardar la información, se ingresa la dirección web de las ofertas, se crea un bucle para que cambie de página una vez extraída la información, finalmente se agrega los elementos del html de los que se extrae la información.

Pre procesamiento de la información

Se le da una estructura de tabla a la información, transformando las listas a un data frame, los datos scrapeados son el precio normal y el precio de oferta, adicional se crea otra columna llamada ahorro que es la resta entre el precio normal y el precio de oferta..

Análisis

En este caso se filtran los datos para buscar los productos que lleven en su descripción la palabra “Queso”, adicional se ordena la información de manera que los productos que generan mayor ahorro se ubiquen primeros en la tabla..
